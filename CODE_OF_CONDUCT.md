# Code of Conduct

## Our Pledge

We want this project to be useful, welcoming, and practical for open-source maintainers.

Contributors are expected to communicate respectfully, assume good intent where reasonable, and keep discussion focused on improving the project.

## Unacceptable Behavior

- harassment or personal attacks
- discriminatory language
- publishing private information without permission
- sustained disruption of project work

## Enforcement

Maintainers may remove inappropriate content, close issues, or block accounts when necessary to protect the project community.
