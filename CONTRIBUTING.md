# Contributing

Thanks for helping improve `oss-maintainer-kit`.

## Good first contributions

- Add checks for another ecosystem file.
- Improve the Codex for Open Source application template.
- Make CLI output clearer for maintainers.
- Add examples from real OSS maintenance workflows.

## Development

Run the local validation command:

```bash
npm run validate
```

The project intentionally avoids runtime dependencies so maintainers can inspect and run it quickly.

## Pull requests

Please keep pull requests focused and include:

- what changed
- why it helps maintainers
- the command you used to validate the change
