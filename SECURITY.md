# Security Policy

## Reporting a Vulnerability

If you find a security issue, please avoid posting exploit details publicly before the maintainer has a chance to respond.

Open a private security advisory on GitHub if available, or contact the maintainer through the repository owner profile.

Please include:

- affected version or commit
- steps to reproduce
- impact
- suggested fix, if known

## Scope

This project is a local CLI and template pack. It does not send repository contents to external APIs.
